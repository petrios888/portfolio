How GPUs work

First off, what is a GPU? GPU stands for Graphics Processing Unit, a GPU is heterogeneous chip mutli-processor for graphics. Imagine gutting out CPU and laying its "internal organs" for specilized computation.

The main underline of a GPu are "Shader cores" which are responsible for Parallel Computing, Parallel computing is the bread and butter of GPUs. Depending on the intitial clock speed and number of shader cores you come up with the number floating point calculations a GPU can do every second.
For example, take the Nvidia GTX 1080 with the number of Shader Cores(Nvidia calls them CUDA cores but same difference.) 2560 and clock speed 1607MHz and the instruction per clock is two becuase GPUs can process insturctions simultaneously . (Number of Cores*Clock Speed*instruction per clock)=FloatingPoint per second .
